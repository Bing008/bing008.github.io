## 个人简介

中共党员，1997年1月生，山东大学电气工程学院博士研究生，师从赵浩然教授。研究方向为**风电精细化实时仿真与优化控制**。

博士期间深度参与全国产化新型电力系统仿真软件 **SimuNPS** 和大规模风电场精细化实时仿真平台 **DigWind** 的研发工作。相关成果被电力系统仿真领域奠基人周孝信院士评价为达到**国际领先水平**。
